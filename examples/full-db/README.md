# my-elit-app

A new Elit project created with create-elit.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3003 to view your app.

## Available Scripts

- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run build:preview` - Build the standalone preview server used for WAPK packing
- `npm run wapk:pack` - Rebuild the preview bundle and pack a fresh WAPK archive
- `npm run wapk:run` - Run the packed WAPK archive
